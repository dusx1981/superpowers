# 语言强度梯度：技术分析

语言强度梯度是结构性权重的底层原理。同一个意图，不同的表达方式，AI 的服从概率截然不同。本系列文档从三层机制出发，分析梯度的来源、效应、策略和极限。

## 核心命题

```
  ┌──────────────────────────────────────────────────────────────┐
  │                                                              │
  │  同一个意思，不同的表达 → AI 的服从概率截然不同             │
  │                                                              │
  │  "建议你先测试"           → 服从概率 ~40%                  │
  │  "你应该先测试"           → 服从概率 ~60%                  │
  │  "你必须先测试"           → 服从概率 ~80%                  │
  │  "NO CODE WITHOUT TEST"   → 服从概率 ~95%                  │
  │  <HARD-GATE>不可写代码</HARD-GATE>  → 服从概率 ~98%        │
  │                                                              │
  │  语义没变（都是"先测试后代码"），信号强度变了。            │
  │  信号强度的来源是什么？这正是梯度分析要回答的。            │
  │                                                              │
  └──────────────────────────────────────────────────────────────┘
```

## 三层机制

语言强度梯度由三层机制叠加产生：

```
  ┌──────────────────────────────────────────────────────────┐
  │                                                          │
  │  Layer 3: 训练数据关联 ← 最深层，最稳定                  │
  │  ══════════════════════                                   │
  │  ALL CAPS / 铁律句式 / 法律语言                          │
  │  在训练语料中与"不可违抗"高度相关                        │
  │                                                          │
  │  Layer 2: 注意力加权 ← 中间层，最直接                    │
  │  ════════════════════                                     │
  │  格式差异（全大写 vs 小写，XML标签 vs 纯文本）           │
  │  在注意力机制中获得不同权重                               │
  │                                                          │
  │  Layer 1: 语用学处理 ← 最表层，最微妙                    │
  │  ══════════════════════                                   │
  │  情态动词 / 否定式 / 全称量词                             │
  │  改变了指令的语用力度                                     │
  │                                                          │
  └──────────────────────────────────────────────────────────┘
```

## 稳健性排序

```
  三层机制的稳健性不同：

  语用层  >  训练层  >  格式层
  最稳定      中等       最脆弱

  - 语用层：情态动词的语义是语言固有的，不依赖模型实现
  - 训练层：依赖训练语料的统计关联，语料变化则效果变化
  - 格式层：依赖 tokenizer 和注意力机制，模型架构变化则效果变化

  推论：如果未来模型的 tokenizer 不再区分大小写，
  格式层会失效，但语用层和训练层仍有效。
  三层全上是防御性设计——不把鸡蛋放在一个篮子里。
```
