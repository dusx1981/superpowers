# 裁量空间：技术分析

裁量空间是贯穿 Superpowers 权重体系的底层概念。结构性权重的全部设计目标：将 AI 的裁量空间从 ~50%（普通文本）压缩到 ~0%（铁律）。本系列从定义、来源、压缩机制、量化模型到极限，完整分析裁量空间。

## 核心定义

```
  ┌──────────────────────────────────────────────────────────┐
  │  裁量空间 = AI 可以自行决定是否遵循指令的自由度。       │
  │                                                          │
  │  它不是一个开关（有/无），而是一个连续的谱：             │
  │                                                          │
  │  裁量空间 = 0%    → AI 没有任何选择，必须服从           │
  │  裁量空间 = 50%   → AI 可以选择服从或不服从             │
  │  裁量空间 = 100%  → AI 完全自由，指令形同虚设           │
  │                                                          │
  │  结构性权重的全部设计目标：                               │
  │  将 AI 的裁量空间从 ~50%（普通文本）                     │
  │  压缩到 ~0%（铁律）。                                    │
  └──────────────────────────────────────────────────────────┘
```
