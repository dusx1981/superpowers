# Controller 提示词完整重构

本文档重构 Controller 在不同平台上收到的完整提示词。

## 提示词构成

Controller 的提示词由**多层嵌套**构成，不是单一文件：

```
  ┌──────────────────────────────────────────────────────────────┐
  │                     Controller 完整提示词                      │
  │                                                              │
  │  ┌─────────────────────────────────────────────────────────┐ │
  │  │ Layer 1: 平台系统提示词                                   │ │
  │  │ 来源：AI 平台自带的 system prompt                        │ │
  │  │ 内容：模型身份、工具定义、安全规则等                       │ │
  │  │ → 不可控，由平台决定                                      │ │
  │  └─────────────────────────────────────────────────────────┘ │
  │                         │                                    │
  │  ┌─────────────────────────────────────────────────────────┐ │
  │  │ Layer 2: 用户指令文件                                     │ │
  │  │ 来源：CLAUDE.md / AGENTS.md / GEMINI.md                  │ │
  │  │ 内容：项目特定的规则和偏好                                 │ │
  │  │ → 优先级最高，覆盖技能和系统提示词                         │ │
  │  └─────────────────────────────────────────────────────────┘ │
  │                         │                                    │
  │  ┌─────────────────────────────────────────────────────────┐ │
  │  │ Layer 3: Superpowers 注入（核心）                         │ │
  │  │ 来源：SessionStart Hook / OpenCode Plugin                │ │
  │  │ 内容：using-superpowers 技能 + 平台工具映射               │ │
  │  │ → 这是 Controller 身份的来源                              │ │
  │  └─────────────────────────────────────────────────────────┘ │
  │                         │                                    │
  │  ┌─────────────────────────────────────────────────────────┐ │
  │  │ Layer 4: 按需技能加载                                     │ │
  │  │ 来源：Skill 工具动态加载                                  │ │
  │  │ 内容：brainstorming / writing-plans / TDD 等              │ │
  │  │ → 按任务需要逐个加载                                      │ │
  │  └─────────────────────────────────────────────────────────┘ │
  │                                                              │
  └──────────────────────────────────────────────────────────────┘
```

## 各平台的注入方式对比

| 平台 | 注入位置 | 注入时机 | 格式 |
|------|---------|---------|------|
| Claude Code | `hookSpecificOutput.additionalContext` | SessionStart Hook | 系统/附加上下文 |
| Cursor | `additional_context` | SessionStart Hook | 附加上下文 |
| Copilot CLI | `additionalContext` | SessionStart Hook | SDK 标准 |
| OpenCode | 首条用户消息 `parts.unshift` | `messages.transform` | 用户消息前缀 |
